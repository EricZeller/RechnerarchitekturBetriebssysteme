Als kleine Vorbereitung führen Sie bitte den folgenden MIPS-Code im WeMIPS-Emulator aus:

addi $v0, $zero, 5
syscall

Sie sollten vom Emulator nach einer Zahl gefragt worden sein. Nach Ausführung des Programms sollte diese Zahl im Register $v0 liegen.

Schreiben Sie ein MIPS-Assemblerprogramm, das so lange nach einer Zahl fragt, bis eine positive Zahl eingegeben wird! Sobald eine positive Zahl eingegeben wurde, soll das Programm beendet werden.

Testen Sie Ihre Implementierung im WeMips-Emulator!

Geben Sie Ihre Implementierung hier an:

# Lösung:

loop:
    addi $v0, $zero, 5
    syscall
    slt $t1, $v0, $zero # Wenn $v0 < 0, setze $t1=1; sonst $t1=0
    bne $t1, $zero, loop # rufe loop nochmal auf wenn t0 ungleich 0 (Zahl war negativ, =1) ist
