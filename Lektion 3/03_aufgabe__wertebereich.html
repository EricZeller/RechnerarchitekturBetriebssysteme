Als kleine Vorbereitung führen Sie bitte den folgenden MIPS-Code im WeMIPS-Emulator aus:

addi $v0, $zero, 5
syscall

Sie sollten vom Emulator nach einer Zahl gefragt worden sein. Nach Ausführung des Programms sollte diese Zahl im Register $v0 liegen.

Schreiben Sie ein MIPS-Assemblerprogramm, das so lange nach einer Zahl fragt, bis die eingebene Zahl im Bereich \left\{0,\ldots,15\right\} liegt. Danach soll das Programm beendet werden.

Testen Sie Ihre Implementierung im WeMips-Emulator!

Geben Sie Ihre Implementierung hier an:

# Lösung:
loop:

    addi $v0, $zero, 5
    syscall
    slti $t0, $v0, 16 # t0 auf 1 wenn v0 kleiner 16
    beq $t0, $zero, end # beenden wenn t0 = 0 (größer 15)
    slt $t0, $v0, $zero # prüfen ob kleiner 0
    beq $t0, $zero, end # beenden wenn t0 = 0 (kleiner 0)

end:
